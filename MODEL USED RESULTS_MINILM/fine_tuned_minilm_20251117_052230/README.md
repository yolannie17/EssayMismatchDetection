# SentenceTransformer based on sentence-transformers/all-MiniLM-L6-v2

This is a [sentence-transformers](https://www.SBERT.net) model finetuned from [sentence-transformers/all-MiniLM-L6-v2](https://huggingface.co/sentence-transformers/all-MiniLM-L6-v2). It maps sentences & paragraphs to a 384-dimensional dense vector space and can be used for semantic textual similarity, semantic search, paraphrase mining, text classification, clustering, and more.

## Model Details

### Model Description
- **Model Type:** Sentence Transformer
- **Base model:** [sentence-transformers/all-MiniLM-L6-v2](https://huggingface.co/sentence-transformers/all-MiniLM-L6-v2) <!-- at revision c9745ed1d9f207416be6d2e6f8de32d1f16199bf -->
- **Maximum Sequence Length:** 256 tokens
- **Output Dimensionality:** 384 dimensions
- **Similarity Function:** Cosine Similarity

### Full Model Architecture

```
SentenceTransformer(
  (0): Transformer({'max_seq_length': 256, 'do_lower_case': False, 'architecture': 'BertModel'})
  (1): Pooling({'word_embedding_dimension': 384, 'pooling_mode_cls_token': False, 'pooling_mode_mean_tokens': True, 'pooling_mode_max_tokens': False, 'pooling_mode_mean_sqrt_len_tokens': False, 'pooling_mode_weightedmean_tokens': False, 'pooling_mode_lasttoken': False, 'include_prompt': True})
  (2): Normalize()
)
```

## Usage

### Direct Usage (Sentence Transformers)

First install the Sentence Transformers library:

```bash
pip install -U sentence-transformers
```

Then you can load this model and run inference.
```python
from sentence_transformers import SentenceTransformer

# Download from the ðŸ¤— Hub
model = SentenceTransformer("sentence_transformers_model_id")
# Run inference
sentences = [
    'Papel ng Sports sa Pagpapaunlad ng Disiplina at Pag-uugali ng Kabataan',
    'Malaki ang ambag ng sports sa pagbuo ng ugali at pagpapanday ng disiplina ng kabataang Pilipino. Sa pagsali sa palakasan, natututo sila ng mahahalagang katangianpaggalang sa patakaran, pagiging responsable, pagtanggap ng tagumpay at pagkatalo, at pakikipagkapwa. Sa ilalim ng Special Program in Sports, ipinaiiral ang mahigpit na retention policies gaya ng minimum na average grade at physical fitness, na humuhubog sa kabataan upang maging masinop sa oras, masigasig sa pag-aaral, at mas malalim ang pang-unawa sa teamwork.? Ang training sa sports ay nagdadala ng self-control, perseverance, at resilience. Natututuhan ng mga mag-aaral na lampasan ang pagod at hamon hindi lang sa laro kundi pati na sa buhay. May mga pag-aaral na nagpapakita ng positibong relasyon ng sports participation sa mabuting asal, mas mataas na antas ng responsibilidad, at pagiging mas bukas sa tamang pamumuhay.? Gayunpaman, isinasabay ng mga kabataan ang sports sa akademikong gawain, kaya mas napapalakas ang kanilang time management at critical thinking skills. Sa kabuuan, ang sports ay epektibong tool para sa pag-unlad ng disiplina, values formation, at kabuuang pagkatao ng kabataan sa Pilipinas.',
    'Ang mga laro sa labas tulad ng taguan at tumbang preso ay bahagi ng ating pagkabata. Nagbibigay ito ng saya at pagkakaisa sa mga bata. Sa kabilang dako, ang linggwistika ay mahalaga upang maunawaan natin ang mga wika at komunikasyon sa paligid. Ito ay pundasyon ng epektibong pakikipag-ugnayan.',
]
embeddings = model.encode(sentences)
print(embeddings.shape)
# [3, 384]

# Get the similarity scores for the embeddings
similarities = model.similarity(embeddings, embeddings)
print(similarities)

#         [0.6746, 1.0000, 0.2635],
#         [0.2164, 0.2635, 1.0000]])
```
